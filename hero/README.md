# Hero

## How to run

You need to have golang installed to run this example.

Install `hero` with the following command:

```
go get github.com/shiyanhui/hero/hero
go get golang.org/x/tools/cmd/goimports
```

Compile the templates:

```
hero -pkgname=main
```

Build the project

```
go build
```

Run the test:

```
./hero
```
